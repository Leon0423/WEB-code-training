下面介紹有關於我想要的鍵盤相關資訊 (官網)

前言 :

我想要一個新的鍵盤作為一份我今年的生日禮物

一方面也是因為現在白色的鍵盤不知為何按下去的阻力很大，筆電的鍵盤又過於平坦，不是非常好施力

剛好看到這副鍵盤身為機械式鍵盤的性價比非常好，但過了折扣時間就會變得不太划算，所以才想提早要求禮物

關於鍵盤 :

  • 最關鍵的價錢 : NT $ 1,199

    從嘖嘖(買賣平台)上才可獲得折扣

  • 期限 : 2024/03/05 02:00

    原則上在這期間是都還可以買，不過數量也是有限的(剩餘 37 份 截至2/25 10:00 )，所以可以的話，希望能早點告訴我結論。

    Ps: 不過還是先看到最後吧，不要到這裡就下結論了。

  • 2.4GHz 無線及有線

    無線接收器收納於鍵盤底部

  • 腳架有3段

    3段式的腳架能更方便的去做調整

  • 充電２小時最長可使用 90天

    附贈一條Type-A to Type-C傳輸線 x 1

  • PBT 鍵帽

    相較於一般ABS的鍵帽，PBT的鍵帽不會很容易油膩及磨損

  • 軸體為熱插拔黃軸

    熱插拔就是說如果某天其中一顆鍵壞掉了，就可以把它拔起來作更換

    原廠有附贈2顆可更換的軸和4個鍵帽
